# DAWN FORM

DAWN SERIES フォーム管理システム

## 機能

- フォーム作成・編集・削除
- フィールドビルダー（テキスト、メール、セレクト、テキストエリア、数値、チェックボックス）
- 公開フォームURL（認証不要）
- 回答収集・閲覧
- ダッシュボード（統計情報）

## 技術スタック

- Next.js (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (認証・DB)
- Playwright (E2Eテスト)

## セットアップ

```bash
npm install
npm run dev
```

デフォルトポート: 3004

## テスト

```bash
npx playwright test
```
